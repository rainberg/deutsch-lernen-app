# 德语学习助手 - 语音转中文功能

## 功能概述

本应用支持将**德语语音**直接转换为**中文翻译**，采用两步处理流程：

```
德语语音 → [Whisper API] → 德语文本 → [GPT API] → 中文翻译
```

## 处理流程

### 步骤1: 语音转写 (Whisper API)
- **输入**: 德语音频文件
- **输出**: 德语文本
- **API**: OpenAI Whisper API (通过 api2d.net)
- **支持格式**: MP3, MP4, M4A, WAV, FLAC, OGG, WEBM

### 步骤2: 文本翻译 (GPT API)
- **输入**: 德语文本
- **输出**: 中文翻译
- **API**: GPT-3.5-turbo (通过 api2d.net)
- **特点**: 保持原文意思和语气，翻译自然流畅

## 使用方法

### 方法1: 在GUI中使用
1. 启动应用: `python main.py`
2. 点击"打开文件"选择德语音频

## 音频要求

### 支持的格式
- MP3 (推荐)
- MP4
- M4A
- WAV
- FLAC
- OGG
- WEBM

### 文件限制
- **最大大小**: 25 MB
- **建议时长**: 不超过25分钟
- **质量**: 清晰的语音，减少背景噪音

### 优化建议
1. **转换格式**: 使用FFmpeg转换为MP3
   ```bash
   ffmpeg -i input.wav -ac 1 -ar 16000 output.mp3
   ```

2. **压缩文件**: 减小文件大小
   ```bash
   ffmpeg -i input.mp3 -b:a 64k output.mp3
   ```

3. **分割长文件**: 使用音频处理器分割
   ```python
   from src.core.audio_processor import AudioProcessor
   
   processor = AudioProcessor()
   segments = processor.split_by_fixed_duration("long_audio.mp3", segment_duration=600)
   ```

## 故障排除

### 问题1: API连接失败
**错误**: `ConnectionError` 或 `Timeout`
**解决**:
1. 检查网络连接
2. 验证API密钥
3. 确认api2d.net服务状态

### 问题2: 文件过大
**错误**: `File size exceeds limit`
**解决**:
1. 分割音频文件
2. 压缩音频质量
3. 转换为更高效的格式

### 问题3: 转写结果为空
**错误**: 空文本返回
**解决**:
1. 检查音频是否有声音
2. 确认语言设置为德语
3. 尝试其他音频文件

### 问题4: 翻译质量差
**症状**: 翻译不准确或不通顺
**解决**:
1. 提高音频质量
2. 使用更清晰的语音
3. 分段处理长文本

## 技术细节

### API调用示例

**Whisper API**:
```python
from openai import OpenAI

client = OpenAI(
    api_key="fk201403-hCEJwCHpGTrUkdzzkvBW93gvrWCM9vDm",
    base_url="https://openai.api2d.net/v1"
)

with open("german_audio.mp3", "rb") as audio:
    response = client.audio.transcriptions.create(
        model="whisper-1",
        file=audio,
        language="de",
        response_format="verbose_json"
    )
    german_text = response.text
```

**GPT翻译**:
```python
response = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[
        {"role": "system", "content": "你是德语-中文翻译助手"},
        {"role": "user", "content": f"请翻译: {german_text}"}
    ]
)
chinese_text = response.choices[0].message.content
```
