# 德语学习助手 - 打包指南

本文档说明如何将德语学习助手打包成：
1. **Windows exe安装包**
2. **安卓 APK安装包**

---

## 📦 Windows 打包 (exe)

### 方案: PyInstaller

#### 前置要求
- Windows 10/11
- Python 3.8+
- 至少 4GB 可用空间

#### 快速打包

**方法1: 使用打包脚本（推荐）**
```cmd
cd C:\deutsch-lernen-app
build\build_windows.bat
```

**方法2: 手动打包**
```cmd
# 1. 安装PyInstaller
pip install pyinstaller

# 2. 安装依赖
pip install -r requirements.txt

# 3. 打包
pyinstaller build/deutsch_lernen.spec --clean
```

#### 输出文件
- 位置: `dist/DeutschLernenApp.exe`
- 大小: 约 100-200 MB (包含Python运行时)

#### 创建安装程序 (可选)

使用 Inno Setup 创建专业的安装程序:

1. 下载安装 [Inno Setup](https://jrsoftware.org/isinfo.php)
2. 打开 `build/setup.iss`
3. 编译生成安装程序

输出: `dist/DeutschLernenApp_Setup_0.1.0.exe`

#### 配置文件说明

**build/deutsch_lernen.spec**
```python
# 主要配置
name='DeutschLernenApp'  # 程序名称
console=False             # 不显示控制台
icon='resources/icons/app_icon.ico'  # 图标
```

#### 常见问题

**Q: 打包后程序很大？**
A: 使用 `--exclude-module` 排除不需要的模块

**Q: 杀毒软件报警？**
A: 这是PyInstaller的常见误报，可添加白名单

**Q: 打包失败？**
A: 检查依赖是否完整安装，查看错误日志

## 📱 安卓打包 (APK)

### 方案: Kivy + Buildozer

#### ⚠️ 重要说明
由于PyQt5不支持安卓，需要使用Kivy重写UI。已提供简化版的Kivy界面 (`mobile/main_mobile.py`)。

#### 前置要求
- Linux (推荐 Ubuntu 20.04+) 或 WSL2
- Python 3.8+
- Java JDK 17
- 至少 10GB 可用空间
- 稳定的网络连接（首次需下载Android SDK）

#### 快速打包

**方法1: 使用打包脚本（推荐）**
```bash
cd /home/deutsch-lernen-app
chmod +x build/build_android.sh
./build/build_android.sh
```

**方法2: 手动打包**
```bash
# 1. 安装系统依赖
sudo apt-get update
sudo apt-get install -y python3-pip openjdk-17-jdk autoconf libtool

# 2. 安装Buildozer
pip3 install buildozer
pip3 install cython==0.29.33

# 3. 编译APK
buildozer android debug
```

#### 输出文件
- 位置: `bin/deutsch_lernen-0.1.0-debug.apk`
- 大小: 约 30-50 MB

#### 首次编译时间
- 下载依赖: 10-20分钟
- 编译Python: 20-30分钟
- 打包APK: 5-10分钟
- **总计**: 约 30-60分钟

#### 生成发布版APK

```bash
# 1. 配置签名信息
# 编辑 buildozer.spec，添加:
# android.release.keystore = /path/to/keystore.jks
# android.release.key.alias = my-key-alias

# 2. 生成发布版
buildozer android release
```

#### 配置文件说明

**buildozer.spec**
```ini
[app]
title = 德语学习助手
package.name = deutsch_lernen
package.domain = org.deutschlernen
version = 0.1.0

requirements = python3,kivy,openai,sqlalchemy,requests
android.permissions = INTERNET,READ_EXTERNAL_STORAGE,WRITE_EXTERNAL_STORAGE,RECORD_AUDIO
android.archs = arm64-v8a,armeabi-v7a
```

#### 安卓版本说明

| 功能 | 完整版(PC) | 安卓版 |
|------|-----------|--------|
| 语音转写 | ✅ | ✅ |
| 翻译 | ✅ | ✅ |
| 单词查询 | ✅ | ✅ |
| 收藏管理 | ✅ | ✅ |
| Anki导出 | ✅ | ⚠️ 有限 |
| GUI框架 | PyQt5 | Kivy |
| 本地模型 | ✅ | ❌ |

#### 常见问题

**Q: 编译失败？**
A: 检查Java版本，确保使用JDK 17

**Q: APK安装失败？**
A: 确保安卓设备允许"未知来源"安装

**Q: 运行闪退？**
A: 查看日志: `adb logcat | grep python`

## 🍎 macOS 打包 (可选)

### 方案: PyInstaller

```bash
# 1. 安装依赖
pip3 install pyinstaller
pip3 install -r requirements.txt

# 2. 打包
pyinstaller build/deutsch_lernen.spec --clean

# 3. 创建DMG
hdiutil create -volname "德语学习助手" -srcfolder dist/DeutschLernenApp -ov -format UDZO dist/DeutschLernenApp.dmg
```

## 🔧 自定义配置

### 更换图标
1. 准备图标文件:
   - Windows: `resources/icons/app_icon.ico` (256x256)
   - 安卓: `resources/icons/app_icon.png` (512x512)
   - 启动画面: `resources/icons/presplash.png` (1080x1920)

2. 更新配置文件中的图标路径

### 减小包大小
1. 排除不需要的模块:
```python
excludes = [
    'tkinter',
    'matplotlib',
    'scipy',
    'jupyter',
]
```

2. 使用UPX压缩:
```python
exe = EXE(..., upx=True)
```

### 添加数据文件
```python
datas = [
    ('config.json', '.'),
    ('resources/*', 'resources'),
]
```
